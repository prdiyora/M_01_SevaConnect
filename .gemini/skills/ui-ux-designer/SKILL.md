# UI/UX Designer Skill

This skill provides design principles and visual standards for the SevaConnect project.

## Design System
- **Font**: Inter (primary).
- **Colors**:
  - Primary: Purple (`#6c3ce0`).
  - Success: Green (`#059669`).
  - Error: Red (`#dc2626`).
  - Warning: Orange (`#f59e0b`).
- **Layout**: Clean, card-based design with ample whitespace.

## Core Workflows

### 1. Visual Design
- Create mockups or descriptions of UI components before implementation.
- Focus on interactive feedback (hover states, loading spinners, transitions).

### 2. Accessibility
- Ensure proper contrast ratios.
- Use semantic HTML tags.

### 3. Consistency Check
- Verify that new components match the existing "SevaConnect" aesthetic.
